# Surfin The Web

A dynamic MVC (Model-View-Controller) application tailored for surfboard enthusiasts. Our platform integrates geocoding technology to provide users with interactive maps, enhancing their browsing experience and facilitating location-based searches for rental options. Utilizing Cloudinary integration, users can effortlessly upload photos, ensuring their listings shine amidst the waves of options. Leveraging the power of Active Record, our robust database management system ensures efficient data handling.

## Access point 🏄🏻

No installation is needed simply click the [**link!**](https://surfboard-rent-70797078df7a.herokuapp.com/)
